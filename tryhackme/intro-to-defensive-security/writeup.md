# Intro to Defensive Security — TryHackMe Writeup

## Overview
This room introduces the concept of defensive security, focusing on how organizations detect, respond, and prevent cyber attacks.

## Key Concepts Learned
- Blue Team vs Red Team
- SOC (Security Operations Center)
- Threat Monitoring & Detection
- SIEM basics
- Incident Response lifecycle

## Important Concepts

### Blue Team
Responsible for defending systems, monitoring logs, and responding to threats.

### SOC
A centralized team that monitors and analyzes security alerts using tools like SIEM.

### SIEM
Security Information and Event Management system used to collect and analyze logs.

## Tools Introduced
- SIEM platforms
- Log monitoring tools
- Alerting systems

## Key Learning
Defensive security is all about visibility. Without logs and monitoring, detecting attacks is nearly impossible.
